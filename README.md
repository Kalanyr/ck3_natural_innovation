# Culture Era Year Requirements Removed For CK3 / Crusader Kings III version 1.18.*+
 1.8. aka kni aka ck3_natural_innovation
A simple mod that removes the year requirement for unlocking Eras.  (Actually sets the year requirement to 0/1/2/3 for Tribal/Early Medieval/High Medieval/Late Medieval , for technical reasons.)  

NB - Paradox made a decent effort at improving the Era unlocking problem disabling Decisions but it's still borked for Varangian Adventurers (which is not added to Norse culture if Tribal and Early Medieval have the same start date), the activity cost increase by era  caused issues with Year 0/1/1/1 setup so that's been changed to 0/1/2/3 ( thank you xziyu6 ) ).
